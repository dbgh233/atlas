Create `app/core/clients/ghl.py` — GHLClient class:
- Constructor takes http_client (httpx.AsyncClient), api_key (str), location_id (str), pipeline_id (str)
- self.base_url = "https://services.leadconnectorhq.com"
- self._headers property returns Authorization, Version: 2021-07-28, Content-Type: application/json
- Methods (all async, all decorated with tenacity @retry):
  - get_opportunity(opp_id: str) -> dict — GET /opportunities/{opp_id}
  - search_opportunities(pipeline_id: str | None = None, limit: int = 100, status: str = "open") -> list[dict] — GET /opportunities/search with location_id, pipeline_id, limit, status params. Handle pagination using startAfter + startAfterId from response sort[] array. Max 5 pages.
  - update_opportunity(opp_id: str, data: dict) -> dict — PUT /opportunities/{opp_id}
  - get_contact(contact_id: str) -> dict — GET /contacts/{contact_id}
  - search_contacts(query: str) -> list[dict] — GET /contacts/?locationId={id}&query={query}
- Retry config: stop_after_attempt(3), wait_exponential(multiplier=1, min=1, max=10), retry on httpx.HTTPStatusError (429 and 5xx only) and httpx.TimeoutException
- Log every retry with structlog: attempt number, wait time, endpoint
- get_ghl_client(request: Request) -> GHLClient dependency function pulling from request.app.state

Create `app/core/clients/calendly.py` — CalendlyClient class:
- Constructor takes http_client (httpx.AsyncClient), api_key (str)
- self.base_url = "https://api.calendly.com"
- self._headers returns Authorization: Bearer {api_key}
- Methods:
  - get_current_user() -> dict — GET /users/me (returns user info including organization URI)
  - list_event_types(organization_uri: str) -> list[dict] — GET /event_types?organization={uri}
  - get_scheduled_event(event_uri: str) -> dict — GET /scheduled_events/{uuid} (parse uuid from full URI)
  - list_webhook_subscriptions(organization_uri: str) -> list[dict] — GET /webhook_subscriptions?organization={uri}&scope=organization
  - create_webhook_subscription(organization_uri: str, callback_url: str, events: list[str]) -> dict — POST /webhook_subscriptions
- Retry: same pattern as GHL (3 attempts, exponential backoff)
- get_calendly_client(request: Request) dependency

Create `app/core/clients/claude.py` — ClaudeClient class:
- Constructor takes client (AsyncAnthropic), model (str = "claude-opus-4-6")

- Log: model, prompt_length on request; input_tokens, output_tokens on response
- get_claude_client(request: Request) dependency
